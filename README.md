# Calculadora de Pensão Alimentícia

Calculadora online para calcular o valor justo da pensão alimentícia.

## 🚀 Deploy

### Vercel (Recomendado)

```bash
# Instale o CLI da Vercel
npm i -g vercel

# Deploy
cd pencao-calc
vercel
```

Ou conecte o repositório no https://vercel.com

### GitHub Pages

1. Faça o push para o GitHub
2. Vá em Settings > Pages
3. Selecione a branch `main` e a pasta `/(root)`
4. O site estará disponível em `https://seu-usuario.github.io/pencao-calc`

## 🛠️ Desenvolvimento

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Acesse: http://localhost:3000
